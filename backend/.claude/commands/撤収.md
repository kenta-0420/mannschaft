# 撤収（開発環境シャットダウン）

以下の手順を順番に実行し、開発環境を安全に終了してください。

## 1. Spring Boot 停止（起動中の場合）

```bash
BOOT_PID=$(lsof -ti:8080 2>/dev/null || true)
if [ -n "$BOOT_PID" ]; then
  kill $BOOT_PID 2>/dev/null && echo "Spring Boot を停止しました (PID: $BOOT_PID)"
else
  echo "Spring Boot は起動していません（スキップ）"
fi
```

## 1-2. Nuxt 停止（起動中の場合）

```bash
NUXT_PID=$(lsof -ti:3000 2>/dev/null || true)
if [ -n "$NUXT_PID" ]; then
  kill $NUXT_PID 2>/dev/null && echo "Nuxt を停止しました (PID: $NUXT_PID)"
else
  echo "Nuxt は起動していません（スキップ）"
fi
```

## 2. Docker コンテナ停止

```bash
wsl -d Ubuntu-24.04 -- bash -c "cd /mnt/c/cloudeProject && docker compose stop"
```

## 3. WSL2 シャットダウン

```bash
wsl --shutdown
```

## 4. 停止確認

WSL2 が停止していることを確認する。

```bash
wsl --list --running
```

「実行中のディストリビューションはありません。」と表示されれば成功。

## 5. 結果報告

全ステップの結果を以下の形式で簡潔に報告する：

| 項目 | 状態 |
|------|------|
| Spring Boot停止 | ✅ / ⏭️（未起動） |
| Docker停止 | ✅ / ❌ |
| WSL2停止 | ✅ / ❌ |

全て ✅ なら「**撤収完了。お疲れさまでございました。**」と報告する。
