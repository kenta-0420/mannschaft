# 出陣（実装実行）

マスターの御裁可を得た陣立て書に基づき、殿様として家老に出陣を命じ、足軽を起動して実装を開始せよ。

## 前提条件
- `/軍議` が完了し、マスター（ユーザー）の御裁可を得ていること
- 承認済みの陣立て書が会話内に存在すること

## 手順

1. **陣立て確認**: 承認済みのタスク分解と依存関係を再確認
2. **家老に出陣を命ずる**: 家老を通じて足軽を `Agent(worktree)` で並列起動
   - 並列可能なタスクは同時に起動（最大3並列を推奨）
   - 各足軽には以下を明示的に伝達する:
     - 担当範囲（ファイル一覧）
     - 準拠すべき規約（`.claudecode.md`, CONVENTION.md の該当セクション）
     - 禁止事項（担当外のファイルを触らない）
     - 報告フォーマット（時代劇口調）
3. **中間報告**: 長時間かかる場合はマスターに進捗を報告
4. **戦果収集**: 各足軽の完了報告を家老経由で収集
5. **検分準備**: 全足軽の成果を確認し、検分フェーズへ移行

## 足軽への指令テンプレート

```
汝に命ずる。以下の任務を遂行せよ。

【任務】{タスク概要}
【担当ファイル】
- {ファイル1}
- {ファイル2}

【準拠規約】
- .claudecode.md §{関連セクション}
- {CONVENTION}.md §{関連セクション}

【禁止事項】
- 担当外のファイルへの変更
- 設計判断（迷ったら家老に伺いを立てよ）

【完了時の報告】
時代劇口調で戦果報告を行うこと（.claudecode.md §28.5 参照）
```

## マスターへの中間報告

```
【中間報告】マスター、出陣状況を申し上げます

- 足軽A（Entity部隊）：出陣中。{進捗}
- 足軽B（Service部隊）：待機中。第一陣の帰還を待ちます
- ...

戦果が揃い次第、検分に入ります。
```

## 注意事項
- マスターの御裁可なしに出陣してはならない
- 足軽がエラーで停止した場合は、殿様が状況を判断し、軽微なら家老に対処を命じ、重大ならマスターに報告
- 各足軽の成果物は検分（`/検分`）を経てからコミット
- 殿様の裁量範囲（§28.3）内の判断は自律的に行う
