## 概要

ADMIN / DEPUTY_ADMIN が組織/チーム内のメンバーへメールを一斉・個別送信する機能。送信対象は全メンバー一斉 / グループ指定 / ロール指定 / 個別選択から選択可能。件名・本文（リッチテキスト）・送信日時（即時 or 予約送信）を設定。チーム/組織単位で週3回まで無料（制限は Redis で管理）。送信履歴・開封率の確認が可能。

**DB 概要（db_design_details.md に定義済み）:**
- `direct_mail_logs`: 送信セッション（scope, subject, body, recipient_type, scheduled_at, status）
- `direct_mail_recipients`: 受信者ごとの配信状況（status: PENDING/SENT/FAILED/BOUNCED, opened_at）
