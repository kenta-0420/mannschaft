## 概要

チャット・投稿・掲示板・ファイルをカード化し、ドラッグ&ドロップで自由配置して重要情報を視覚的にストック・整理する機能。個人/チーム/組織の3スコープ対応。

**ボードのスコープ:**
- 個人ボード: 自分専用（非公開、複数作成可能）
- チーム共有ボード: チームメンバー全員閲覧可能（作成は ADMIN/DEPUTY_ADMIN）
- 組織共有ボード: 組織メンバー全員閲覧可能

**カードの種類:**
- 参照カード: チャットメッセージ・タイムライン投稿・掲示板スレッド/返信・ブログ記事・ファイルをストック（`content_snapshot` で内容保持）
- メモカード: ボード専用の独立テキストメモ
- URL カード: 外部 URL + OGP サムネイル
- 見出しカード: 横長タイトルバー（視覚的ラベル）

**セクション（折りたたみグループ）:**
- 見出し付き領域にカードを格納し、折りたたみ（collapse）で縮小可能
- 見出しカードとの違い: セクションはカードを格納できる、見出しカードは視覚的な仕切り線

**ビジュアル整理:**
- 自由配置（X/Y 座標を DB 保存）
- カラーラベリング（6色）
- カードサイズ（small / medium / large）
- ボード背景（コルク風テクスチャ / ホワイト / ダーク）

**コンテキスト保持:**
- 参照カードクリック → サイドパネルで元コンテンツの前後5件を表示
- `content_snapshot` により元の投稿が削除されてもカード上でテキスト確認可能

**追加機能:**
- 各カードに付箋メモ（`user_note`）追記可能
- カードごとに自動アーカイブ日時（`auto_archive_at`）設定可能
- ボード内キーワード検索・カラーフィルター

**Phase 1〜8 は `FEATURE_V9_ENABLED=false` により UI・API を完全非表示。DB テーブルは Phase 1 で先行作成。**
