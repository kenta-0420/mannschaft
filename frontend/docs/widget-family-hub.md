## 概要

個人ダッシュボードに「家族」ウィジェットを追加する。  
`template === 'FAMILY'` のチームに所属しているユーザーに対して、各家族チームの **お知らせ** と **未着手TODO** をまとめて表示する。  
複数の家族チームに所属している場合は、タブ切り替えで各家族の情報を確認できる。

---

## 表示条件

| 条件                                                          | 挙動                                         |
| ------------------------------------------------------------- | -------------------------------------------- |
| `teamStore.myTeams` に `template === 'FAMILY'` が **1件以上** | ウィジェットにコンテンツを表示               |
| 家族チームが **0件**                                          | 「家族チームに参加していません」空状態を表示 |

## UI仕様

### ウィジェットカード

| 項目             | 値                        |
| ---------------- | ------------------------- |
| タイトル         | 家族                      |
| アイコン         | `pi pi-home`              |
| グリッド幅       | `col-span-2`（2カラム分） |
| 手動リフレッシュ | あり                      |

### 家族タブ（複数家族所属時）

- 家族チームが **2件以上** の場合、上部にタブボタンを表示
- 選択中 → プライマリカラー背景・白文字
- 未選択 → サーフェスカラー背景
- **1件のみ** → タブ非表示。ファミリー名をサブテキストとして表示

### コンテンツレイアウト

```
┌──────────────────────────────────────────────────────────────────┐
│ [家族Aタブ]  [家族Bタブ]                        (複数所属時のみ) │
│                                                                  │
│ ─── お知らせ ──────────────  │  ─── TODO ─────────────────────  │
│  📌 [緊急] タイトル         │    ○ TODO タイトル             │
│     3分前                   │      期限: 明日                    │
│  タイトル                   │    ○ TODO タイトル              │
│     2時間前                 │                                    │
│                 すべて表示 → │                      すべて表示 →  │
└──────────────────────────────────────────────────────────────────┘
```

### お知らせセクション仕様

| 項目         | 仕様                                                      |
| ------------ | --------------------------------------------------------- |
| 取得件数     | 最新4件                                                   |
| APIメソッド  | `getScopedThreads('teams', teamId, { page: 0, size: 4 })` |
| ピン留め     | `isPinned === true` のとき 📌 アイコン表示                |
| 優先度バッジ | `priority !== 'INFO'` のとき表示（下表）                  |
| 表示項目     | タイトル・相対時間                                        |
| リンク先     | `/teams/{teamId}/bulletin`                                |

**優先度バッジ**

| priority    | ラベル       | 配色                                        |
| ----------- | ------------ | ------------------------------------------- |
| `CRITICAL`  | 緊急         | 赤（`bg-red-100 text-red-700`）             |
| `IMPORTANT` | 重要         | オレンジ（`bg-orange-100 text-orange-700`） |
| `WARNING`   | 注意         | 黄（`bg-yellow-100 text-yellow-700`）       |
| `INFO`      | （表示なし） | —                                           |

### TODOセクション仕様

| 項目        | 仕様                                                        |
| ----------- | ----------------------------------------------------------- |
| 取得件数    | 最大4件                                                     |
| APIメソッド | `listTodos('team', teamId, { status: 'PENDING', size: 4 })` |
| フィルター  | `status === 'PENDING'`（APIパラメータで指定）               |
| 表示項目    | タイトル・期限（`dueDate` がある場合のみ相対時間で表示）    |
| リンク先    | `/teams/{teamId}/todos`                                     |

## データソース

| データ         | composable                                                     | エンドポイント                                          |
| -------------- | -------------------------------------------------------------- | ------------------------------------------------------- |
| 家族チーム一覧 | `useTeamStore().myTeams`（`template === 'FAMILY'` でフィルタ） | `GET /api/v1/me/teams`                                  |
| お知らせ       | `useBulletinApi().getScopedThreads`                            | `GET /api/v1/teams/{id}/bulletin/threads?page=0&size=4` |
| TODO           | `useTodoApi().listTodos`                                       | `GET /api/v1/teams/{id}/todos?status=PENDING&size=4`    |
