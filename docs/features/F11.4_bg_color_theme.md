# F11.4 背景色テーマ機能

## 機能概要

ライトモード時の背景色をユーザーが10色のプリセットから選択できる機能。
ダークモード時は選択色を無効化し、PrimeVue デフォルトの `bg-surface-ground` を使用する。

---

## 実装方針

| 要素 | 詳細 |
|---|---|
| CSS変数 | `--bg-color`（`documentElement` に設定） |
| 状態管理 | Pinia store（`useAppearanceStore`） |
| 永続化 | localStorage（プライマリ）+ `/api/v1/settings/appearance`（セカンダリ） |
| モード制御 | ライトモード時のみ `--bg-color` を適用。ダークモード切替時に変数を削除 |

---

---

## ダークモード時の挙動

- `applyBgColor()` が呼ばれた際、`isDark === true` の場合は `--bg-color` CSS変数を削除する
- `applyTheme()` の末尾でも `applyBgColor()` を呼ぶことで、テーマ切替時に即座に背景色制御が反映される
- `default.vue` レイアウトは `dark:bg-surface-ground` クラスを持ち、ダークモード時はPrimeVue標準の背景色を使用する

---

## 設定の永続化

1. **localStorage（プライマリ）**: `appearance` キーに JSON 形式で保存
2. **サーバー（セカンダリ）**: `/api/v1/settings/appearance` に非同期 PUT
   - 失敗してもサイレントフォールバック（localStorageが正として動作）
